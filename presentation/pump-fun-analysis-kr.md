# Pump.fun: 10억 달러 규모의 Memecoin 공장

### Solana 최대 논란의 프로토콜에 대한 종합 분석

---

## 1. 요약

Pump.fun은 Solana 기반의 토큰 런치패드로, 누구나 2달러 미만의 비용으로 수 초 만에 memecoin을 배포할 수 있게 하여 무허가(permissionless) 토큰 생성의 패러다임을 바꿔 놓았다. 2024년 1월 출시 이후, 10명 미만의 직원으로 운영되는 이 플랫폼은 **누적 매출 10억 2천만 달러**를 달성했으며, **1,750만 개의 토큰**을 발행하고, Solana에서 memecoin 시장 점유율 최대 **98%**를 차지했다. 이후 자체 DEX(PumpSwap)를 출시하며 수직 통합을 이루었고, **13억 2천만 달러 규모의 ICO**를 단 12분 만에 완료했다. 그러나 이 플랫폼은 여전히 깊은 논란의 중심에 있다: 연구에 따르면 **출시된 토큰의 98%가 사기**이며, **수익을 낸 사용자는 0.4%에 불과**하고, 프로토콜은 해킹, 소송, 규제 조사 등에 직면해 왔다. Pump.fun은 암호화폐 분야에서 무허가 혁신과 소비자 보호 사이의 긴장을 보여주는 대표적인 사례다.

## 2. Pump.fun이란?

### 개요

Pump.fun은 Solana 기반의 **무허가 토큰 런치패드**로, 새로운 토큰을 생성하고 거래하는 데 필요한 진입 장벽을 획기적으로 낮추었다. Pump.fun 이전에는 토큰 발행에 기술적 지식, 유동성 공급, DEX 상장 조율 등이 필요했다. Pump.fun은 이 모든 과정을 단일 트랜잭션으로 축소시켰다.

### 주요 정보

| 항목 | 세부 내용 |
|---|---|
| **출시일** | 2024년 1월 |
| **설립자** | 영국 출신 기업가 (익명) |
| **블록체인** | Solana |
| **직원 수** | 10명 미만 |
| **발행 토큰 수** | 약 1,750만 개 |
| **누적 매출** | 10억 2천만 달러 이상 |
| **토큰 생성 비용** | 약 0.02 SOL (약 $2) |

### 작동 방식 — 간략 흐름도

```
┌─────────────┐     ┌──────────────────┐     ┌─────────────────┐     ┌──────────────┐
│  생성자가     │────>│  Pump.fun이       │────>│  Bonding Curve   │────>│  DEX로        │
│  토큰을      │     │  Solana에 토큰    │     │  거래 단계       │     │  Graduation   │
│  발행        │     │  배포             │     │  (~$69K 시가총액) │     │  (PumpSwap)  │
└─────────────┘     └──────────────────┘     └─────────────────┘     └──────────────┘
       │                     │                        │                       │
       │              1% 생성 수수료            1% 거래 수수료         Migration 수수료
       │                     │                        │                       │
       └─────────────────────┴────────────────────────┴───────────────────────┘
                                         │
                                    Pump.fun
                                    수익으로 귀속
```

### Pump.fun 토큰의 생애주기

1. **생성** — 사용자가 이름, 티커, 이미지, 설명을 설정하면 Solana에서 토큰이 발행된다.
2. **Bonding Curve 단계** — 토큰이 내부 AMM(자동화 시장 조성자)에서 거래된다. SOL이 예치될수록 수학적으로 정의된 커브를 따라 가격이 상승한다.
3. **Graduation** — Bonding curve가 채워지면 (약 85 SOL 예치, 약 $69K 시가총액), 유동성이 자동으로 DEX로 이전된다.
4. **공개 시장** — 토큰이 PumpSwap(이전 Raydium)에서 완전한 유동성과 함께 자유롭게 거래된다.

## 3. Bonding Curve의 작동 원리

### 수학적 모델

Pump.fun은 **virtual reserves**를 활용한 **constant-product bonding curve** (x × y = k)를 사용하여 시작 가격이 0이 되지 않도록 보장한다. 이는 수학적으로 Uniswap V2의 CPAMM 공식과 동일하지만, 근본적으로 다른 맥락에서 적용된다. 양쪽에 실제 자산이 예치된 실제 유동성 풀 대신, pump.fun은 실제로 존재하지 않는 "가상" SOL로 시딩된 합성 풀을 생성한다. 이를 통해 모든 토큰이 실제 유동성 없이도 명확하게 정의된 가격 커브 위에서 거래를 시작할 수 있다.

#### 초기 파라미터

| 파라미터 | 값 |
|---|---|
| **Virtual SOL Reserve** | 30 SOL |
| **Virtual Token Reserve** | 1,073,000,000 토큰 (10억 7,300만) |
| **총 토큰 공급량** | 1,000,000,000 토큰 (10억) |
| **커브 내 실제 토큰** | 약 793,100,000 토큰 |
| **Constant Product (k)** | 30 × 1,073,000,000 = 32,190,000,000 |
| **Graduation 기준** | 실제 예치 약 85 SOL (가상 약 115 SOL) |

#### 가격 공식

Bonding curve의 임의 시점에서:

```
Price = Virtual SOL Reserve / Virtual Token Reserve

시작 가격:
  P₀ = 30 / 1,073,000,000 = 0.00000002796 SOL per token

Δ 토큰 매수 시 비용:
  Cost (in SOL) = k / (token_reserve - Δ) - sol_reserve
  
  여기서:  k = sol_reserve × token_reserve
```

#### 가격 변화 예시

| 실제 예치 SOL | Virtual SOL | 잔여 토큰 | 토큰당 가격 (SOL) | 대략적 시가총액 |
|---|---|---|---|---|
| 0 | 30 | 1,073,000,000 | 0.0000000280 | $4,000 |
| 10 | 40 | 804,750,000 | 0.0000000497 | $7,100 |
| 25 | 55 | 585,272,727 | 0.0000000940 | $13,400 |
| 50 | 80 | 402,375,000 | 0.0000001989 | $28,400 |
| 85 | 115 | 279,913,043 | 0.0000004109 | $69,000 |

#### 핵심 인사이트

> 이 커브는 시작부터 graduation까지 **약 14.7배의 가격 상승**을 만들어낸다. 초기 매수자는 현저히 유리한 가격을 확보할 수 있어 강한 FOMO 역학이 형성된다. Virtual reserves 덕분에 가격이 절대 0이 되지 않으며, 0으로 나누는 오류를 방지하고 최초 매수자도 항상 실질적인 가격을 지불하게 된다.

#### 모든 토큰이 동일한 가격에서 Graduation하는 이유

모든 토큰이 동일한 파라미터 — 같은 virtual reserves, 같은 공급량, 같은 커브 형태 — 로 시작하기 때문에, graduation에 도달하는 모든 토큰은 정확히 같은 가격(약 0.000000411 SOL/토큰)과 같은 시가총액(약 $69K)에서 졸업한다. 이는 의도적인 설계 선택으로, 복잡성을 제거하고 진행 상황을 보편적으로 비교 가능하게 하며, 커스텀 커브 파라미터를 통한 조작을 방지한다.

#### Graduation 시 가격 불연속성

중요한 미묘한 점이 있다: virtual_sol = 30 파라미터는 graduation 시점의 가격 교란을 최소화하도록 정밀하게 계산되었다. 수수료가 0인 경우, bonding curve 가격과 PumpSwap 풀 가격은 거의 동일하다 (virtual token reserve인 1,073,000,000은 수학적 최적값과 43,231 토큰 차이에 불과하다). 그러나 pump.fun은 migration 수수료로 약 6 SOL을 가져가기 때문에, graduation 시점에 약 7%의 가격 하락이 발생한다. 이 갭은 설계 결함이 아닌 의도적인 비즈니스 결정으로, 프로토콜이 graduation을 수익화하는 방식이다.

#### Graduation 시 일어나는 일

약 85 실제 SOL이 예치되면:
- 약 7억 9,310만 토큰이 커브에서 매수된 상태 (사용자가 보유)
- 약 2억 690만 토큰이 남아있으며 — 이 토큰들은 **bonding curve에 포함된 적이 없다.** 생성 시점부터 LP 시딩을 위해 별도로 확보된 것이다.
- 약 79 SOL (수수료 차감 후) + 2억 690만 예비 토큰 → PumpSwap LP 풀로 이전
- **LP tokens가 소각된다** — 유동성이 영구적으로 잠기며, 그 누구도 (pump.fun도, 생성자도, 누구도) 인출할 수 없다. 이는 DeFi에서 가장 흔한 rug pull 수법인 LP 인출을 원천 차단한다.

> **참고:** LP token 소각은 유동성 제거를 방지하지만, 생성자의 토큰 덤핑, 내부자 sniping, 공모 pump-and-dump 등 다른 사기를 방지하지는 **않는다.**

## 5. 수직 통합: PumpSwap

### 수익 유출에서 수익 포획으로

Pump.fun 역사상 가장 전략적으로 중요한 행보는 2025년 3월 **PumpSwap** 출시였다. PumpSwap 이전에는 bonding curve를 졸업한 모든 토큰이 Raydium으로 이전되었다. 이는 graduation 이후의 모든 거래 수수료(성공적인 토큰의 경우 종종 가장 수익성이 높은 단계)가 경쟁사로 흘러갔음을 의미한다.

```
BEFORE (2024년 1월 – 2025년 3월):
  Pump.fun (토큰 생성) ──graduation──> Raydium (DEX 거래)
  
  수익: Bonding curve 수수료만 (시가총액 약 $0-69K 구간)

AFTER (2025년 3월 – 현재):
  Pump.fun (토큰 생성) ──graduation──> PumpSwap (자체 DEX)
  
  수익: Bonding curve 수수료 + 모든 graduation 이후 거래 수수료
```

### 통합 수수료 전체 그림

![pump.fun Ecosystem Combined Daily Fees](../analysis/charts/combined_ecosystem_fees.png)

이 차트에서 몇 가지 핵심 인사이트가 도출된다:

**1. PumpSwap이 수익 모델을 변혁했다.** 2025년 4월 이전에는 모든 수익이 bonding curve 수수료(주황색)에서만 발생했다. PumpSwap 출시 후 두 번째 수익 레이어(파란색)가 등장했다. 2025년 후반에는 PumpSwap 수수료가 특정 날에 bonding curve 수수료와 대등하거나 이를 초과하는 경우가 빈번했다.

**2. 누적 수치가 압도적이다.** 요약 박스에 따르면 bonding curve 수수료 9억 8,600만 달러, PumpSwap 수수료 5억 4,500만 달러로, 합산 **총 생태계 수수료 15억 3,000만 달러**에 달한다. PumpSwap만으로도 약 1년 운영 만에 5억 달러 이상을 창출했다.

**3. 수익 다각화가 회복탄력성을 만든다.** 2025년 10월경의 2차 피크(합산 일일 약 850만 달러)는 새로운 토큰 발행이 아닌 DEX 거래 활동에 의해 주도되었다. 이는 pump.fun이 신규 토큰 발행 속도가 둔화되더라도 상당한 수익을 창출할 수 있음을 의미하며, 수익이 전적으로 신규 런칭에 의존하던 이전 모델에서 중대한 진화를 이루었다.

**4. 안정 상태가 인상적이다.** 2026년 초 기준, 합산 일일 수수료는 약 300~400만 달러 수준에서 bonding curve와 DEX 수수료가 대략 균등한 비율로 안정화되었다. 이 연환산 10억 달러 이상의 런레이트는 — 10명 미만의 팀에서 — 여전히 경이로운 수준이다.

PumpSwap의 순 거래량은 급속한 도입 후 시장 정상화의 과정을 보여준다:

![PumpSwap DEX Daily Volume](../analysis/charts/pumpswap_volume.png)

PumpSwap은 출시 후 첫 두 달 만에 **일일 거래량 4~6억 달러**에 도달했다 — 수직 통합에 대한 pump.fun의 베팅을 입증하는 놀라운 도입 속도였다. 총 누적 거래량은 약 13개월 만에 **689억 달러**에 달했다. 7일 이동평균은 2025년 5월에 일일 약 6억 달러로 정점을 찍은 후 하반기에 걸쳐 점진적으로 하락했다. 주목할 만한 특징으로는 2026년 2월에 약 9억 달러에 달하는 대규모 단일일 스파이크(바이럴 memecoin 이벤트에 의한 것으로 추정)와 2026년 초 일일 5,000만~1억 달러 수준으로의 안정화가 있다.

정점 대비 하락이 반드시 약세를 의미하지는 않는다 — 이는 memecoin 광풍의 전반적인 냉각을 반영한다. 일일 5,000만~1억 달러의 일관된 기본선은 진정한 지속적 수요를 나타내며, 플랫폼은 내재적 해자를 보유하고 있다: pump.fun의 bonding curve를 졸업하는 모든 신규 토큰이 자동으로 PumpSwap에 상장되어, 새로운 거래 페어가 지속적으로 유입된다.

## 8. 토큰 결과의 현실

### 출시 후 토큰에 실제로 일어나는 일

데이터는 냉정한 현실을 보여준다. 아래 차트는 플랫폼에서 29개 토큰의 실시간 샘플을 분석하여, 시가총액 분포와 상위 성과자를 보여준다:

![pump.fun Live Token Analysis](../analysis/charts/live_token_analysis.png)

이 스냅샷은 pump.fun 생태계의 잔혹한 파워 로(power law) 역학을 포착한다:

**좌측 패널 (시가총액 분포):** 샘플링된 29개 토큰 중 **10개(34%)의 시가총액이 $10K 미만**으로 사실상 죽었거나 방치된 상태였다. 추가로 6개가 $10K~$50K 구간, 4개가 $50K~$100K 구간에 위치했다. 합산하면 **토큰의 68% 이상이 $100K 미만의 가치**를 가졌다. 수백만 달러를 돌파한 것은 극소수에 불과했고, $100M을 초과한 것은 하나도 없었다. 이 분포는 전체 플랫폼 통계에서 1~1.7%의 토큰만이 graduation에 성공한다는 수치와 일치한다.

**우측 패널 (시가총액 상위 15개):** 극심한 편중이 즉시 눈에 띈다. 샘플에서 가장 큰 토큰이 나머지 모든 것을 완전히 압도하여, 나머지 14개 "상위" 토큰은 같은 스케일에서 거의 보이지 않는다. 이것이 전형적인 파워 로다 — 극소수의 토큰이 거의 모든 가치를 차지하고, 대다수는 무가치하다.

### Graduation 퍼널

```
17,500,000 개 토큰 생성
     │
     ▼
   175,000 – 297,500 졸업 (1.0 – 1.7%)
     │
     ▼
   약 17,500 시가총액 $1M+ 도달 (약 0.1%)
     │
     ▼
   약 175 $10M+ 이상 유지 (<0.001%)
     │
     ▼
   약 10-20개가 "블루칩" memecoin으로 성장
```

| 지표 | 수치 | 함의 |
